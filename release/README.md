# PeekPairs Release

`PeekPairs.app.zip` is the current packaged macOS app. Download it, unzip it, and open `PeekPairs.app`.

The archive is regenerated by:

```sh
scripts/build-release.sh
```

That script also updates `/Applications/PeekPairs.app` on this Mac unless `PEEKPAIRS_SKIP_INSTALL=1` is set.
